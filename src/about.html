About Scratchpad

This is a web-based text editor which implements:

  • rich text support using contenteditable
  • custom formatting options such as bold, italics, underline, with different font styles and font sizes

UI/UX features an intuitive, aesthetic dark theme, with a customised scrollbar.
Languages used are JavaScript, HTML and CSS.
